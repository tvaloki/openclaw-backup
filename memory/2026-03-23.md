# 2026-03-23 - Recovery & Stabilization

## Key Learnings & Decisions
- **Memory Hierarchy**: Canonical memory is stored in external Postgres (Supabase). Local files like `MEMORY.md` and `BOOTSTRAP.md` are optional summaries. Do not infer amnesia from their absence.
- **Git Auth**: GitHub has deprecated password authentication. Use SSH keys (recommended) or Personal Access Tokens (PAT).
- **Git Branching**: Older local repos may still use `master`, but GitHub defaults to `main`. Rename local branch or push explicitly to resolve `refspec` errors.
- **Model Availability**: 
    - The gateway can cache auth failures as `auth_permanent`, preventing retries even after config fixes. A gateway restart (`openclaw gateway restart`) is required to flush this state.
    - Google Gemini is currently the only viable primary provider; OpenRouter key is dead (401) and OpenAI Codex is rate-limited.
- **Telegram Routing**: If `groupPolicy` is `allowlist`, an empty `groupAllowFrom` will cause the gateway to drop 100% of group traffic.
